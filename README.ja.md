# kidssafe-natsume

福井市棗（なつめ）地区における子どもたちのための地域安全マップを作成する、Code for Fukuiによるシビックテックプロジェクトです。地域の懸念箇所を特定して地図上にマッピングし、子どもたちの安全向上を図ります。

## デモ

https://code4fukui.github.io/kidssafe-natsume/

## 機能

-   **インタラクティブマップ**: 「こども110番の家」や潜在的な危険箇所などの地域安全情報を、カスタムアイコンで表示します。
-   **詳細情報**: マップ上のピンをクリックすると、懸念事項の内容や改善案などの詳細情報をデータから直接確認できます。
-   **データ駆動型**: すべてのマップポイントは、編集が簡単なCSVファイルで管理されます。
-   **高い適応性**: 設定ファイルとデータファイルを変更するだけで、他の地域にも簡単に再利用できます。バックエンドは不要です。
-   **正確な位置情報**: [Geo3x3](https://geo3x3.com/) を使用し、地理座標を効率的かつ正確にエンコードして共有します。

## ご自身の地域への適用方法

本プロジェクトは、任意の地域に合わせて簡単に設定できるよう設計されています。

1.  **フォークまたはクローン**: リポジトリのコピーを取得します。
2.  **メタデータのカスタマイズ**: `index.html` を編集し、メインタイトル（`<title>`）やヘッダーのテキストを変更します。
3.  **レイヤーの設定**: `index.csv` を編集してマップのレイヤーを定義します。各行でレイヤー名、データファイル、表示アイコンを紐付けます。
4.  **データの追加**: 地域の安全データを含むCSVファイルを作成または編集します（例: `kidssafe/` ディレクトリ内）。詳細は「データ構造」セクションを参照してください。
5.  **アイコンの追加**: カスタムマップアイコンを `icon/` ディレクトリに配置します。
6.  **公開**: GitHub Pagesなどのホストに静的サイトをデプロイします。

## 開発

本プロジェクトは静的ウェブサイトであり、ビルドステップは不要です。ローカルでのテストには、モジュールのインポートを正しく処理するためのシンプルなWebサーバーが必要です。

[Deno](https://deno.land/) を使用すると、1つのコマンドでWebサーバーを起動できます。

```bash
# プロジェクトのルートディレクトリから実行します
deno run --allow-net --allow-read https://deno.land/std/http/file_server.ts
```

## データ構造

マップの設定およびデータ入力は、CSVファイルを使用して行います。

### メイン設定（`index.csv`）

このファイルでは、マップ上に表示されるデータレイヤーを定義します。

| カラム | 説明 | 例 |
| :----- | :----------------------------------------------------------- | :----------------------- |
| `name` | データレイヤーの表示名 | `100当番の家` |
| `fn` | 位置データを含むCSVファイルへのパス | `house100.csv` |
| `icon` | このレイヤーのアイコンファイル名（`icon/`ディレクトリ内） | `house110_icon.png` |

### 位置データ（例: `kidssafe/田ノ頭町.csv`）

これらのファイルには、レイヤーごとの個別のポイントデータが含まれます。

-   最初の行でヘッダーを定義します。これらのヘッダーは、マップ上のポップアップ情報ボックスのフィールドラベルとして使用されます。
-   マップ上にポイントを配置するため、`Geo3x3` 列が必須です。

**例（`kidssafe/田ノ頭町.csv`）:**

```csv
危険箇所・気がかりな点,改善案,Geo3x3
田ノ頭町内のため池。柵は設置されているが、転落等の注意が必要。,,E913873765598238
```
