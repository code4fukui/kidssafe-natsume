# 福井市 棗地区 地域安全マップ「キッズセーフ」

福井市棗地区の地域安全マップ「キッズセーフ」のリポジトリです。子供の安全を守るため、地域の危険箇所を特定し、マップ化したものです。

## デモ
- https://code4fukui.github.io/kidssafe-natsume/

## 機能
- 地域の危険箇所をマップ上に表示
- 詳細情報（場所、危険要因、改善提案など）を確認できる
- 新たな危険箇所の登録や情報の更新が可能

## 必要環境
特になし。ウェブブラウザがあれば利用可能です。

## 使い方
1. デモサイトを開く
2. メニューボタンからマップ上の危険箇所を確認
3. 新たな危険箇所を発見した場合は、データ更新方法に従って情報を登録

## データ・API
危険箇所の情報は、CSVファイルで管理しています。Geo3x3の座標情報を使用して、マップ上に表示しています。
