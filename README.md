# kidssafe-natsume

A civic tech project for a community safety map for kids in the Natsume district of Fukui City, Japan.

## Demo
https://code4fukui.github.io/kidssafe-natsume/

## Features
- Interactive map with pinned locations of safety concerns
- Ability to edit and contribute new safety locations
- Reusable across different districts with customization

## Requirements
- [Deno](https://deno.land/) for local development

## Usage
1. Clone the repository
2. Customize the title, data, and other details for your local district
3. Publish the site using GitHub Pages

## Data
The map data is stored in CSV files, such as `house100.csv` and `kidssafe/田ノ頭町.csv`.
